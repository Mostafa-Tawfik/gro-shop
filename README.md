## Run Locally

Clone the project

```bash
  git clone https://github.com/Mostafa-Tawfik/gro-shop.git
```

Go to the project directory

```bash
  cd gro-shop
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
